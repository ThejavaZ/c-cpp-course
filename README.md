# 🚀 C & C++ Course: De Cero a Programación de Sistemas

¡Bienvenido! Este es mi repositorio personal de aprendizaje de **C y C++**. El objetivo de este curso es asentar y profundizar mis conocimientos en estos lenguajes fundamentales, enfocándome en los fundamentos de la programación de sistemas, algoritmos eficientes y el desarrollo de aplicaciones de alto rendimiento y bajo nivel.

> [!IMPORTANT]
> **Colaboración IA:** Este curso ha sido estructurado y desarrollado en colaboración con **Gemini**, actuando como tutor técnico para asegurar las mejores prácticas de la industria y una transición fluida entre paradigmas.

## 🚀 Estructura del Proyecto

El curso está organizado en módulos incrementales, diseñados para ser consultados como una referencia rápida en el futuro:

### ⚙️ Fundamentos de C

Esta sección explora los conceptos básicos y avanzados del lenguaje C.

- **c/01-hola.c**: Tu primer programa en C.
- **c/02-variables.c**: Declaración y uso de variables.
- **c/03-constantes.c**: Definición y manejo de constantes.
- **c/04-capturar.c**: Entrada de datos por teclado.
- **c/05-imprimir.c**: Salida de datos en consola.
- **c/06-tipos.c**: Exploración de tipos de datos básicos.
- **c/07-strings.c**: Manejo de cadenas de caracteres.
- **c/08-condicionales.c**: Estructuras de control condicionales (if, else if, else, switch).
- **c/09-loops.c**: Bucles (for, while, do-while).
- **c/10-listas.c**: Introducción a arreglos (arrays).
- **c/11-operaciones-listas.c**: Operaciones comunes con arreglos.
- **c/12-operaciones-string.c**: Más operaciones con cadenas.
- **c/13-funciones.c**: Creación y uso de funciones.

### ➕ Introducción a C++ y Programación Orientada a Objetos

Esta sección cubre la transición a C++ y los pilares de la Programación Orientada a Objetos.

- **cpp/01-hola.cpp**: Tu primer programa en C++.
- **cpp/02-variables.cpp**: Variables en C++.
- **cpp/03-constantes.cpp**: Constantes en C++.
- **cpp/04-capturar.cpp**: Entrada de datos (cin).
- **cpp/05-imprimir.cpp**: Salida de datos (cout).
- **cpp/06-tipos.cpp**: Tipos de datos en C++.
- **cpp/07-strings.cpp**: Manejo de `std::string`.
- **cpp/08-condicionales.cpp**: Condicionales en C++.
- **cpp/09-loops.cpp**: Bucles en C++.
- **cpp/10-listas.cpp**: Introducción a `std::vector` y arreglos.
- **cpp/11-operaciones-listas.cpp**: Operaciones con contenedores.
- **cpp/12-operaciones-strings.cpp**: Más operaciones con `std::string`.
- **cpp/13-funciones.cpp**: Funciones y sobrecarga.
- **cpp/14-clases.cpp**: Definición y uso de clases y objetos.
- **cpp/15-encapsulacion.cpp**: Principios de encapsulación.
- **cpp/16-herencia.cpp**: Implementación de herencia.
- **cpp/17-polimorfismo.cpp**: Polimorfismo y funciones virtuales.
- **cpp/18-modulos.cpp**: Introducción a módulos y namespaces.
- **cpp/19-errores.cpp**: Manejo de excepciones.
- **cpp/elements/persona.h**: Ejemplo de archivo de cabecera.
- **cpp/ejercicios/**: Directorio para ejercicios prácticos.

## 📖 Cómo usar este repo

Cada carpeta contiene archivos `.c` o `.cpp` comentados que explican el "por qué" de cada sintaxis. Para compilar y ejecutar cualquier módulo:

### Para archivos C:

```bash
cd c/
gcc archivo.c -o programa_c
./programa_c
```

### Para archivos C++:

```bash
cd cpp/
g++ archivo.cpp -o programa_cpp
./programa_cpp
```
